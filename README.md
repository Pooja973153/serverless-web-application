# Serverless Web Application using AWS Lambda

## Project Overview

This project demonstrates the implementation of a serverless web application using AWS Lambda, API Gateway, DynamoDB, and IAM.

The application processes user requests without managing any servers by using AWS serverless services.

---

## Project Objective

To build a scalable, cost-effective serverless application using AWS managed services.

---

## AWS Services Used

- AWS Lambda
- API Gateway
- DynamoDB
- IAM

---

## Architecture

User

↓

API Gateway

↓

AWS Lambda

↓

DynamoDB

---

## Features

- Serverless Architecture
- Event Driven Execution
- No Server Management
- Automatic Scaling
- Cost Effective

---

## Learning Outcome

This project helped me understand serverless computing, API integration, IAM permissions, and NoSQL database management using DynamoDB.
